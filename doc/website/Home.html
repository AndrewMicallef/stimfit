Stimfit

 
 

Stimfit is a free, fast and simple program for viewing and analyzing electrophysiological data. Its user interface is based on Stimfit for DOS, a PASCAL program written by Peter Jonas (Physiological Institute, University of Freiburg). Stimfit is currently available for GNU/Linux, Mac OS X and Windows. It features an embedded Python shell that allows extending the program functionality using numerical libraries such as NumPy and SciPy.

Supported file types include:

Read/write

        CFS binary data

        HDF5 files

        Axon text files

        ASCII files

Read-only

        Axon binary files (*.abf) version 1 and 2 (pClamp 6-10)

        Axograph files (*.axgd, *.axgx)

Write-only

        Igor binary waves (*.ibw)


Support for other file types may be implemented upon request.
