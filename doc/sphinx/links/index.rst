===============================
Free electrophysiology software
===============================
`RELACS <http://relacs.sourceforge.net/>`_ 
    Software for data acquisition, analysis, and stimulus generation. Runs under GNU/Linux.
`WinLTP <http://www.ltp-program.com/indexWinLTP.htm>`_ 
    Stimulation and data acquisition program for Windows, specialised for LTP/LTD experiments. 
`NClamp <http://www.physiol.ucl.ac.uk/research/silver_a/nclamp/index.html>`_ 
    Plugins for IgorPro for data acquisition. Igor is available for Mac and Windows.
`Neuromatic <http://www.neuromatic.thinkrandom.com/index.html>`_ 
    Plugins for IgorPro for data analysis. Igor is available for Mac and Windows.
`Strathclyde Electrophysiology Software <http://spider.science.strath.ac.uk/sipbs/page.php?show=software_ses>`_ 
    Suite of programs for recording and analysing signals from intracellular electrophysiology experiments. Runs under Windows.

================================
Electrophysiology software sites
================================
`Neurohost <http://www.neurohost.org>`_ 
    José Guzmán's neuroscience web site. Maintains up-to-date instructions how to build and install stimfit on GNU/Linux (see Troubleshooting).
`Neuronal Networks Lab, University of Nottingham <http://www.nottingham.ac.uk/neuronal-networks/links3.html>`_ 
    Quite a number of useful links to electrophysiology software sites.
